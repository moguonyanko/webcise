CSS Variables

CSS Variablesの読み書き

CSS Variablesはスクリプトで読み書きできる

CSS Variablesを使うことでスタイルに分かりやすい名前を付けることができる。
変数の値はスクリプトで参照することができ，変更することもできる。

CSS Variablesの値を確認する。(変数名： 変数の値)
スクリプトで変数の値を変更するとページ内の全ての要素に影響がある。
スクリプトで変数の値を変更した後は変更後の値が得られる。

CSS Variablesの上書き

CSS Variablesは上書きできる。

変数を上書きしてもdocumentElementから得られる値は変化せず，
ルート要素に対して変数上書きを行っても変化しない。

子要素は親要素での変数上書きの影響を受ける。

子要素でなければ変数上書きの影響を受けない。
