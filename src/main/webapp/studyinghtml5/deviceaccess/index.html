Device Access API

様々なデバイスが発生させるイベントやそれを扱うAPIについてまとめます。

DOM3 Events

主にフォーカスやキーボードに関するイベントを扱うためのAPIが定義されている。

Focus Events

似たイベントが定義されているがバブリングの有無が異なっている。 イベントリスナーは各form要素に対して登録している。 form要素からイベントが伝搬しない場合，イベントリスナーが処理されない。

focusinとfocusoutはバブリングあり
focusとblurはバブリングなし

Input Events

input要素やtextarea要素の内容に変更が加えられた時にInputEventが発生する。 ただし同じテキストをコピー・アンド・ペーストしてもInputEventは発生する。 テキストが空になった時も発生する。

Keyboard Events

キーボード操作時に発生する。ただしデバイスによっては発生しない可能性がある。 テキスト入力に対して何らかの処理を行いたいならばInputEventを利用するべきである。
