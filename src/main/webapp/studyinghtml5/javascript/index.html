ビューポート(ウインドウ)のサイズ

WindowオブジェクトのinnerWidthプロパティとinnerHeightプロパティから ビューポート(ウインドウ)のサイズを取得する。
ここでいうビューポート(ウインドウ)とは表示領域のことである。

ウインドウサイズ結果

コンテナのサイズ

ElementのoffsetWidth, offsetHeightにはボーダーと(表示されていれば)スクロールバーの サイズが含まれる。
ElementのclientWidth, clientHeightにはそれらは含まれない。
従ってスクロールバーが表示されるとそのElementのclientWidth, clientHeightはCSSで指定されたサイズよりも 小さくなる。
ElementのscrollWidth, scrollHeightにはElementが領域からはみ出している部分を
clientWidth, clientHeightに加算した値が保持される。
はみ出していなければclientWidth, clientHeightと同じ値になる。

コンテナサイズ結果

コンテナのテキストのサイズ

インライン要素のclientWidth, clientHeightは常にゼロである。
インライン要素のサイズを得たければElementのoffsetWidth, offsetHeightを使う。
ボックス要素同様offsetWidth, offsetHeightにはボーダーのサイズが含まれる。
インライン要素なのでスクロールバーを考慮する必要は無い。
インライン要素が複数行で表示されている時，offsetHeightにはその表示状態を
反映したサイズが保持されている。つまり行数が増えるほどoffsetHeightが増す。
複数行で表示されていても1行で表示されている時のサイズが得たいときは
ElementのscrollWidth, scrollHeightを使い，必要に応じボーダーのサイズを加算する。
